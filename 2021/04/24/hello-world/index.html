open-ptc

总体介绍
PTC code 是Particle Tracer Code for Energetic Particles in Tokamak的简称，是基于托卡马克几何的三维非圆截面，包含全轨道和导心漂移轨道的混合测试粒子程序。PTC code 采用读取 gfile (eqdsk) 平衡磁场，追踪粒子，模拟托卡马克位形电磁场扰动引起的高能粒子损失。

FUsion Simulation and Modelling Alliance Platform，聚变模拟与建模联合平台 (FUSMAP)

为积极发展面向磁约束核聚变等离子体的集成模拟平台软件，链接更多集成模拟技术与人才资源，深化交流与合作，对平台的数据标准化和集成策略、模拟平台的开放性与可扩展性、社区建设与培训、目标与路线图等议题进行深入研讨，加速推动“磁约束集成模拟开放平台”的搭建，经国内两次（2023年9月，2024年4月）集成模拟研讨会的讨论，拟开始开展”集成模拟开放平台建设“，本网站用于本开放平台公共信息的收集和传播。

集成模拟开放平台介绍

An integrated modeling and experimental data analysis software for magnetically confined thermonuclear fusion experiments.
The goal is to enhance existing scientific workflows and enable new integrated modeling capabilities.

联系人：Feng Wang, Huasheng Xie, Jian Liu, Zhi Yu
