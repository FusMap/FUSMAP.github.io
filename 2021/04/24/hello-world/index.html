open-ptc

总体介绍
PTC code 是Particle Tracer Code for Energetic Particles in Tokamak的简称，是基于托卡马克几何的三维非圆截面，包含全轨道和导心漂移轨道的混合测试粒子程序。PTC code 采用读取 gfile (eqdsk) 平衡磁场，追踪粒子，模拟托卡马克位形电磁场扰动引起的高能粒子损失。

聚变模拟与建模联合平台(FUSMAP)介绍

FUSMAP,读作fuse map，全称FUsion Simulation and Modelling Alliance Platform，聚变模拟与建模联合平台。
面向磁约束核聚变等离子体中的快速建模和物理模拟，以开放合作的方式联合不同单位、个人等方式发展开放的磁约束等离子体物理集成模拟平台。

在开发阶段FUSMAP将先逐步提供如下基本功能：

  1. 针对磁约束核聚变等离子体物理的相关物理模拟模块
  2. 磁约束等离子体实验基础开放数据
